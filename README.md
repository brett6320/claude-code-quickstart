# macOS Dev Quickstart

Setup guides, order matters — later steps assume earlier ones done.

1. [Homebrew](01-homebrew.md) — package manager, foundation for rest
2. [VS Code](02-vscode.md) — editor
3. [Other recommended tools](03-other-tools.md) — git, iTerm2/Warp, Node, gh CLI, etc.
4. [Claude Code guardrails](04-guardrails.md) — baseline safety rules for Claude Code
5. [Caveman skill](05-caveman.md) — token-efficient terse mode for Claude Code

Run 1→3 once per machine. 4→5 are Claude Code config, redo per machine (or sync via dotfiles).
