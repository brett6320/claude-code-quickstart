# macOS Dev Quickstart

Setup guides for a new Mac. Written for anyone — no assumed experience with Terminal or command-line tools. Each guide explains what a tool is, why you'd want it, and the exact steps.

**What's "Terminal"?** It's an app on your Mac for typing text commands instead of clicking. Some tools below need it because they don't have an app you install by clicking — you'll be told exactly when and how to open it, and exactly what to type.

**How to use the command boxes in these guides:** when you see text in a gray box like this:

```bash
echo "hello"
```

that means: open Terminal, type or paste that exact line, then press Return/Enter. Don't type the `$` symbol if you see one — that's just a convention some docs use to mark "this is a command," not something you type.
