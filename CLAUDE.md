# claude-code-quickstart

This repo is a set of human-facing setup guides (01–05) for a new Mac, plus
`SETUP.md` — a one-shot, Claude-Code-ingestible file that installs the
guardrails + caveman-mode hooks described in guides 04–05 automatically.

If the user asks to "set up guardrails," "set up caveman mode," or similar in
this repo, read `SETUP.md` and execute it top to bottom. Don't run it
unprompted.
